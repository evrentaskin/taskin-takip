1. ZIP içeriğini mevcut projenin kök klasörüne kopyalayın.
2. Dosyaların üzerine yazılmasına izin verin.
3. CMD/Terminal:
   npm install
   npm run build
   git add .
   git commit -m "V10.0 özel dersler ve profil düzeltmeleri"
   git push

SQL çalıştırılması gerekmez. Özel ders verileri mevcut shared_app_state tablosunda saklanır.
