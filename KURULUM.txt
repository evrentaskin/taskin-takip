V9.6.2 ONLINE DENEME BEYAZ EKRAN DÜZELTMESİ

Sorun:
Supabase save_my_science_online_attempt RPC fonksiyonu güncellenen tek denemeyi nesne olarak döndürürken öğrenci paneli bunu tüm deneme listesi sanıp state'e yazıyordu. Ardından exams.filter çalışınca "filter is not a function" hatası oluşuyor ve ekran beyaza dönüyordu.

Düzeltme:
- RPC'nin tek deneme nesnesi dönüşü mevcut deneme listesine güvenli biçimde birleştirildi.
- Eski RPC sürümlerinin dizi döndürmesi de desteklenmeye devam ediyor.
- localStorage veya buluttan hatalı tip gelirse öğrenci ekranının çökmesini önleyen dizi kontrolleri eklendi.
- Deneme başlatma ve cevap kaydetme sonrası aktif sınav güvenli biçimde güncelleniyor.

Kurulum:
1. ZIP içindeki src klasörünü proje klasörünün üzerine kopyalayın.
2. npm run build
3. git add .
4. git commit -m "Online deneme beyaz ekran duzeltmesi"
5. git push
