V9.6.0 FEN YILLIK PLAN YAMASI

1) ZIP içindeki src ve public klasörlerini mevcut projenin üzerine kopyalayın.
2) Terminalde çalıştırın:
   npm install
   npm run build
3) Build başarılı olunca:
   git add .
   git commit -m "Fen yıllık plan özelliği"
   git push

Vercel GitHub'a bağlıysa push işleminden sonra otomatik yayınlar.
SQL çalıştırılması gerekmez.

ÖZELLİKLER
- Sol menüde Yıllık Plan, Oturma Planı'nın hemen üstünde.
- Ayarlar içinde 5, 6, 7 ve 8. sınıf için ayrı Excel yükleme/güncelleme/silme.
- Kullanılmayan sınıfın planını yüklemek zorunlu değildir.
- Excel şablonu uygulamadaki Ayarlar ekranından indirilebilir.
- Ekran her açıldığında bugünün tarihine ait haftayı gösterir.
- Oklarla Excel'deki tüm önceki ve sonraki haftalarda sınırsız gezinilir.
- Bugünkü Haftaya Dön düğmesi bulunur.
- Aynı haftaya birden fazla kazanım satırı eklenebilir.
- Veriler mevcut shared_app_state altyapısında bulutta saklanır.
