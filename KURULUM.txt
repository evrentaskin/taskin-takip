TAŞKIN TAKİP V9.6.1 — ONLINE DENEME DOSYA EKİ

1) ZIP içindeki src klasörünü projenin üzerine kopyalayın.
2) Supabase > SQL Editor bölümünde şu dosyayı çalıştırın:
   supabase/migrations/20260719_online_exam_files.sql
3) Terminalde:
   npm install
   npm run build
   git add .
   git commit -m "Online deneme PDF ve resim eki"
   git push

ÖZELLİKLER
- Fen ve LGS online denemelerine PDF/JPG/PNG/WEBP eklenebilir.
- En fazla dosya boyutu 20 MB'dir.
- Öğrenci dosyayı yalnızca denemenin başlangıç-bitiş süresi içinde açabilir/indirebilir.
- Öğrenci cevabı kesin kaydedince dosya erişimi kapanır.
- Süre bitince dosya düğmeleri görünmez; yalnızca sonuç ve analiz kalır.
- Öğretmen tekrar çözdürürse erişim sınav süresi hâlâ devam ediyorsa yeniden açılır.
- Bağlantılar 2 dakikalık imzalı URL olarak oluşturulur.

BUILD KONTROLÜ
- npm run build başarıyla tamamlandı.
- Vite yalnızca mevcut büyük bundle uyarısını verdi; derleme hatası yoktur.
