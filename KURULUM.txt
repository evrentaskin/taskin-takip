V9.5.3 - ÖĞRENCİ PANELİ DURUM PDF BEYAZ SAYFA DÜZELTMESİ

DÜZELTİLEN BÖLÜM:
- Yalnızca 5, 6, 7 ve 8. sınıf öğrenci panelindeki Denemelerim > Durum PDF.
- Öğretmen paneli ve öğretmen PDF'leri değiştirilmemiştir.

KAYNAK KODLA KURULUM:
1. ZIP içindeki src klasörünü çalışan projenin üzerine kopyalayın.
2. Dosyaları değiştirmeyi onaylayın.
3. Projenin ana klasöründe CMD açın.
4. npm run build
5. Test için npm run dev

HAZIR DIST İLE KURULUM:
- Mevcut dist klasörünü ZIP içindeki dist klasörüyle değiştirin.
- CMD çalıştırmanız gerekmez.

SQL ve npm install gerekmez.
