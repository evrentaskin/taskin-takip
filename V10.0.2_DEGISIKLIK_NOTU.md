# V10.0.2 Öğrenci Profili, Akıllı Oturma ve LGS PDF Düzeltmesi

- Ayarlar içindeki profil alanları tek **Öğrenci Profili** bölümünde toplandı.
- Profil alanlarında yazı, sayı, tarih, telefon, seçim listesi ve evet/hayır veri türleri korunur.
- Yeni profil alanları tüm öğrenci profillerinde görünür.
- Tanıma Raporu PDF/Excel alan seçiminde sonradan eklenen profil alanları da görünür.
- Oturma Planı akıllı dağıtımında profil alanları seçilebilir.
- Yan yana oturma seçenekleri: Serbest, bir kız-bir erkek, kızlar yan yana, erkekler yan yana.
- Çalışkan-destek eşleştirme ve mevcut hazır kurallar korunur.
- Boş bırakılan koltuk sürükle-bırak ve dokunarak yerleştirmeyi kabul etmez; küçük ikon ile açılıp kapanır.
- LGS online deneme PDF soru analizi tek sayfaya sığacak şekilde sıkılaştırıldı.
- SQL gerekmez.
- npm run build başarıyla tamamlandı.
