# V10.0.3 — Öğrenci Profili, Etiket, Oturma Planı ve LGS PDF Düzeltmesi

## Yapılanlar

- Öğrenci profilindeki **Oturma Bilgileri / Öğrenci Tanıma** sekmeleri kaldırıldı.
- Bütün profil alanları tek **Öğrenci Profili** ekranında birleştirildi.
- Ayarlar > Öğrenci Profili bölümünde hazır ve sonradan eklenen tüm etiketlere silme düğmesi eklendi.
- Etiket türleri: Yazı, Sayı, Tarih, Telefon, Seçim listesi, Evet/Hayır.
- Yeni profil alanları tüm öğrencilerde, Tanıma Raporu PDF/Excel seçiminde ve Oturma Planı kural listesinde görünür.
- Eski `student_information_cards` ve `student_profile_tags` kayıtları kaybedilmeden yeni ortak şemaya alınır.
- Oturma Planı profil alanları, kalabalığı azaltmak için tikli çoklu seçim açılır listesine dönüştürüldü.
- Boş bırakılan koltuklarda yazı kalabalığı kaldırıldı; küçük yasak simgesi kullanıldı. İşaretli koltuk sürükle-bırak ve akıllı dağıtım kabul etmez.
- LGS Online Deneme PDF soru analizi sayfası, tek yatay A4 sayfaya sığması için sıkılaştırıldı.

## Kurulum

Bu ZIP içeriğini proje klasörünün üzerine kopyalayın ve dosyaların değiştirilmesine izin verin.

```bash
npm install
npm run build
```

SQL çalıştırmak gerekmez.
