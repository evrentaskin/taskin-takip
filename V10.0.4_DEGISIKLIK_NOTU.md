# V10.0.4 — Avatar, Oturma Planı ve Tanıma Raporu

## Öğrenci avatarı
- Cinsiyet değişince uygun kız/erkek avatar havuzundan otomatik avatar seçilir.
- Gözlük seçimi değişince aynı saç modelinin gözlüklü/gözlüksüz karşılığına geçilir.
- Kız ve erkek öğrenciler için sekizer saç/yüz çeşidi ve bunların gözlüklü sürümleri kullanılır.
- Yeni avatar gerektiğinde, aktif öğrenciler arasında en az kullanılan uygun avatar tercih edilir.
- Avatar sabit dairesel çerçevede ortalanır; görsel değişirken kayma ve taşma olmaz.
- Ayrı "Avatarı Yenile" düğmesi kaldırılmıştır.

## Oturma planı
- "Öğrenci profili verileri" açılır seçim kutusu kaldırılmıştır.
- Hazır akıllı dağıtım kuralları bilgileri doğrudan öğrenci profilinden okur.
- LGS modülünde değişiklik yapılmamıştır.

## Öğrenci Tanıma Raporu
- Hazır alanlarla profil etiketlerinin aynı bilgiyi iki kez göstermesi engellenmiştir.
- Numara ve tekilleştirilmiş öğrenci profili alanları aynı listede gösterilir.
- Sonradan eklenen profil alanları PDF ve Excel sütun seçiminde otomatik görünür.
- PDF yalnızca yatay A4 hazırlanır.
- Yazı ve hücre ölçüleri öğrenci/sütun sayısına göre otomatik sıkıştırılır.
- Sınıfın tamamı tek sayfaya sığdırılacak şekilde çıktı oluşturulur.

## Kurulum
Yama klasörünün içindeki `src` ve `public` klasörlerini proje ana klasörünün üzerine kopyalayın.

Ardından:

```bash
npm install
npm run build
git add .
git commit -m "V10.0.4 avatar oturma ve tanima raporu"
git push
```

SQL çalıştırılması gerekmez.
