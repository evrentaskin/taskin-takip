# V10.0.5 — Performans, Mobil Oturma Planı ve Başlangıç Sayfası

- Oturma planında her sütunun sıra sayısı telefonda `− / +` düğmeleriyle değiştirilebilir.
- Sıra sayısı en az 1, en fazla 12 olacak şekilde korunur.
- Sayı alanını silince tekrar 1'e dönme sorunu ortadan kaldırıldı.
- Uygulama yeni açıldığında ve tarayıcı geri yüklenen sayfayı açtığında başlangıç sayfası daima **Ana Sayfa** olur.
- Bunun için Ayarlar'a ek bir seçenek eklenmedi.
- Büyük uygulama sayfaları React `lazy` ve `Suspense` ile ihtiyaç duyulduklarında yüklenir; ilk açılış paketinin gereksiz modülleri beklemesi azaltıldı.
- LGS işlevlerine dokunulmadı.

## Kurulum
ZIP içindeki `src` klasörünü proje ana klasöründeki `src` klasörünün üzerine kopyalayın.

```bash
npm install
npm run build
git add .
git commit -m "V10.0.5 performans mobil oturma ve anasayfa"
git push
```
