# Taşkın Takip Sistemi V6.4.3

- Ana sayfadaki Ayın Öğrencileri kartı Raporlar modülüyle aynı aylık puan kurallarını kullanır.
- Puan listesi iki saniyede bir, pencere yeniden odaklandığında ve tarayıcı depolama değiştiğinde canlı yenilenir.
- Öğrenciler ekranına Sınıf Giriş Bilgileri düğmesi eklendi.
- Aktif sınıfın öğrenci numarası, adı, kullanıcı adı ve bu cihazda kayıtlı şifresi topluca görüntülenebilir ve kopyalanabilir.
- Supabase Auth eski şifreleri geri döndürmediği için yalnızca bu cihazda oluşturulan veya yenilenen şifreler gösterilir.
- Kayıtlı olmayan şifreler öğrenci satırındaki Şifre Yenile işlemiyle yeniden belirlenebilir.
- npm run build başarıyla tamamlandı.
