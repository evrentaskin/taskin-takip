# V6.4 LGS Öğretmen Ana Sayfası

Bu paket mevcut Taşkın Akademi projesi korunarak güncellendi.

Eklenenler:
- Son deneme ortalama neti ve öğrenci sayısı kartları
- Normal deneme ve online deneme kartları
- Normal/online deneme listeleri
- Son 7 deneme çizgi grafiği
- Son deneme ders ortalamaları grafiği
- En fazla yükselen/düşen öğrenci kartları
- LGS geri sayımı ayarı
- Mevcut Excel yükleme ve Supabase deneme kaydetme akışının korunması

Çalıştırma:
1. npm install
2. npm run dev
