# V8 Mobil ve Tablet Uyumluluk

Bu sürümde:
- Telefon ve tablet için responsive yerleşim
- Mobil çekmece menü
- Dokunmatik kullanım için en az 44px kontrol alanları
- Telefonlarda tek sütun kart düzeni
- Tabletlerde iki sütun düzeni
- Geniş tablolar için yatay kaydırma
- Tam ekran ve küçük ekran modal düzeni
- iPhone/iPad güvenli alan desteği
- PWA manifest ve service worker
- Ana ekrana eklenebilme desteği

eklenmiştir.

## Yerel çalıştırma
npm install
npm run dev

## İnternetten kullanım
Projeyi Vercel/Netlify gibi HTTPS sunucusuna yayınlayın. PWA ve ana ekrana ekleme HTTPS üzerinde çalışır.
