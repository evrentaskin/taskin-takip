V10.0.6.4 ÖZEL DERS GÖRSEL VE PDF DÜZELTME YAMASI

Yapılanlar:
- Analiz sekmesinde öğrenci düzenle butonu gizlendi.
- Boş/beyaz çıkan analiz PDF'i için PDF oluşturma alanı ve html2canvas ayarları düzeltildi.
- Takvim ve ücret bölümü iki sütunlu yapıldı: solda ücret özeti, sağda takvim.
- Solda yapılan ders saati, ödenen ders ücreti, ödenmeyen ders ücreti ve toplam borç gösteriliyor.
- Takvimde yapılan ders yeşili koyulaştırıldı, yazı okunurluğu artırıldı.
- Takvim günlerinin içinde Ödendi / Ödenmedi bilgisi eklendi.
- Seçili öğrenci kartı turuncu çerçeve ve hafif turuncu zeminle belirginleştirildi.
- Seçili öğrenci panelinde çerçeveler turuncu yapıldı.

Kurulum:
ZIP içindeki src/pages/PrivateLessonsPage.jsx dosyasını ana projedeki aynı dosyanın üzerine kopyalayın.
Ardından:
npm install
npm run build
git add .
git commit -m "V10.0.6.4 özel ders görsel ve PDF düzeltmeleri"
git push

Build kontrolü başarıyla tamamlandı.
