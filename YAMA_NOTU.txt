V10.0.6.5 ÖZEL DERS SEÇİM, RENK VE PDF YAMASI

Yapılanlar:
- Seçilen öğrencinin kartının tamamı turuncu yapıldı; isim, kullanıcı adı ve simgeler beyaz gösteriliyor.
- Takvimdeki yeşil ve kırmızı renklerin tonu azaltıldı, yazılar koyu ve okunaklı hale getirildi.
- Öğrenci düzenleme butonu yalnızca üstteki öğrenci kartında, öğrenci adının yanında bırakıldı.
- Seçili öğrenci detay başlığındaki ikinci Düzenle butonu kaldırıldı.
- Beyaz/boş PDF sorunu için PDF alanı negatif katmandan çıkarıldı ve html2canvas oluşturma ayarları yenilendi.
- PDF oluştururken hata olursa kullanıcıya uyarı gösteriliyor.

Kurulum:
ZIP içindeki src/pages/PrivateLessonsPage.jsx dosyasını ana projedeki aynı dosyanın üzerine kopyalayın.
Ardından:
npm install
npm run build
git add .
git commit -m "V10.0.6.5 özel ders seçim renk ve PDF düzeltmesi"
git push
